## Projetos e experimentos

Tenho trabalhado em diferentes projetos envolvendo IA, automações, Supabase, APIs, sistemas web e ferramentas digitais.

Um dos projetos em desenvolvimento é o **PedeSampa**, uma iniciativa voltada para soluções digitais para negócios locais de delivery, como restaurantes, pizzarias, hamburguerias e marmitarias.

A proposta do PedeSampa é explorar formas mais simples e acessíveis de criar presença digital, organizar pedidos e aproximar pequenos negócios de seus clientes.
